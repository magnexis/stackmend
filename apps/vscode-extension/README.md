# STACKMEND

Repository intelligence for diagnosing integration fractures, tracking repair outcomes, and learning from software maintenance workflows.

## Current Commands

- `StackMend: Scan Project`
- `StackMend: Generate Project Truth`
- `StackMend: Start Daemon`
- `StackMend: Stop Daemon`
- `StackMend: Restart Daemon`
- `StackMend: Mark Repair Successful`
- `StackMend: Mark Repair Incorrect`
- `StackMend: Correct This Diagnosis`

## What The Extension Does

STACKMEND focuses on the places where systems stop agreeing with each other:

- frontend routes vs backend routes
- repository diagnostics vs verified outcomes
- repair suggestions vs developer corrections
- daemon lifecycle vs persistent runtime health

The extension uses native VS Code surfaces such as commands, diagnostics, and daemon-backed workflows instead of a heavy custom dashboard.
