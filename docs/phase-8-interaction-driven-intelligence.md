# Phase 8 - Interaction-Driven Intelligence

## Mission

Phase 8 makes every meaningful interaction with StackMend contribute to measurable intelligence change.

The governing law is:

> No meaningful interaction is disposable. Every result, correction, success, failure, and verification must leave StackMend more accurate, more capable, or more aware of its limitations than it was before.

## Core loop

```text
Interaction
-> observe evidence
-> compare prediction with outcome
-> extract learning signal
-> update scoped memory
-> verify the update
-> use improved knowledge next time
```

## Phase goals

- every meaningful interaction produces an Interaction Learning Record
- every important interaction produces an Intelligence Delta
- confirmed mistakes become replayable regressions
- repeated mistakes become StackMend self-defects
- competence is measured across sessions instead of assumed from database growth

## Acceptance direction

Phase 8 is only real if StackMend can show that an equivalent future task is handled:

- more accurately
- with fewer attempts
- with better confidence
- with lower regression risk
- without repeating confirmed mistakes
