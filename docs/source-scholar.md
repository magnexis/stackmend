# Source Scholar

## Purpose

`Source Scholar` is the approved-source study subsystem for STACKMEND.

It should ingest:

- official documentation
- official repositories
- maintainer-authored examples
- permissively licensed educational repositories
- verified coding challenge sources
- standards and specifications

## Source policy

Every source adapter should record:

- source name
- acquisition method
- license verification
- automation permission
- attribution requirements
- rate-limit policy

If permission cannot be confirmed, STACKMEND should not use the source.

## Anti-goals

- arbitrary scraping
- private solution harvesting
- account impersonation
- bypassing source restrictions
