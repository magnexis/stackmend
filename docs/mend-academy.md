# Mend Academy

## Purpose

`Mend Academy` is STACKMEND's internal autonomous training institution.

It should continuously:

- assign problems
- run attempts in sandboxes
- score correctness and performance
- compare against verified references when legitimately available
- extract reusable techniques
- retest weak areas
- feed improvements back into repository repair behavior

## Daily cycle

Suggested autonomous cycle:

```text
Refresh approved source metadata
-> import newly permitted exercises and examples
-> revalidate changed framework techniques
-> run weakness-targeted curriculum
-> execute mutation-repair tournament
-> retake failed challenges
-> perform transfer tests
-> evaluate candidate Code Reflexes
-> compact learning records and decay stale knowledge
```

## Examination classes

- technique examinations
- broken-repository repair examinations
- mutation-repair tournaments
- transfer examinations
- adversarial examinations

## Safety requirements

- no unauthorized platform automation
- no uncontrolled scraping
- no host execution of arbitrary learned code
- no promotion without verification and evaluation
